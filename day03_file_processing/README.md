# Day 03：Python 文件处理

## 学习目标

本项目用于练习 Python 文件处理基础，包括：

- 读取 TXT 文件
- 写入 TXT 文件
- 读取 JSON 文件
- 写入 JSON 文件
- 读取 CSV 文件
- 使用 try-except 处理异常
- 将 JSON 问答数据转换成 Markdown 笔记
